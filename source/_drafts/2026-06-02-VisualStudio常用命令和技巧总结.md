如果你说的是 Microsoft Visual Studio（简称 VS，而不是 VS Code），那么：

删除当前行：

* `Ctrl + L`：直接删除当前行（光标放在该行任意位置即可）

这是很多人最常用的编辑快捷键之一。

```text
# 快捷键  — 注释为功能
Ctrl + L               # 删除当前行
Ctrl + X               # 剪切当前行（未选中文本时）
Ctrl + C               # 复制当前行（未选中文本时）
Ctrl + Z               # 撤销
Ctrl + Y               # 重做
Ctrl + F               # 查找
Ctrl + Shift + F       # 全局查找
Ctrl + H               # 替换
Ctrl + Shift + H       # 全局替换
Ctrl + G               # 转到指定行
Ctrl + K, Ctrl + C     # 注释代码
Ctrl + K, Ctrl + U     # 取消注释
Ctrl + K, Ctrl + D     # 格式化整个文档
Ctrl + K, Ctrl + F     # 格式化选中代码
Alt + F12              # 快速查看定义
F12                    # 跳转到定义
Ctrl + -               # 返回上一个位置
Ctrl + Shift + -       # 前进到下一个位置
Ctrl + R, Ctrl + R     # 重命名变量/函数
Ctrl + Shift + B       # 构建项目
F5                     # 启动调试
Ctrl + F5              # 不调试运行
F10                    # 单步跳过
F11                    # 单步进入
F9                     # 设置断点
```

另外推荐记住这些高频快捷键：

如果你其实用的是 Visual Studio Code（VS Code），删除当前行则是：

* Windows/Linux：`Ctrl + Shift + K`
* macOS：`Cmd + Shift + K`
