滑动窗口

思想和双指针的思想是类似的

方法 1:暴力一个控制区间的起始位置一个控制区间的终止位置,循环遍历
> nums.size() <= 10^5,暴力的时间复杂度是O(n^2),会超时

```cpp
class Solution {
public:
    int minSubArrayLen(int target, vector<int>& nums) {
        int res = INT_MAX;
        for(int i = 0;i<nums.size();++i){
            int sum = 0;
            for(int j = i;j<nums.size();++j){
                sum += nums[j];
                if(sum >= target){
                    res = min(res,j - i + 1);
                    break;
                }
            }
        }
        return res == INT_MAX ? 0 : res ;
    }
};
```

方法 2:滑动窗口
```cpp
class Solution {
public:
    int minSubArrayLen(int target, vector<int>& nums) {
        int i = 0,res = INT_MAX,sum = 0;
        for(int j = 0;j < nums.size();++j){
            sum += nums[j];
            while(sum >= target){
                res = min(res,j - i + 1);
                sum -= nums[i];
                ++i;
            }
        }
        return res == INT_MAX ? 0 : res ;
    }
};
```
