1. 安装 vsce 打包工具（只需一次）

```bash
npm install -g @vscode/vsce
```

这里以 acwing-vscode-plugin 插件为例。

2. 在项目目录打包

```bash
cd D:\code\acwing-vscode-plugin
npm run compile
vsce package
```

这会在项目根目录生成一个 .vsix 文件（例如：vscode-acwing-1.0.0.vsix）。感觉本质上就是一个压缩包。

![打包截图1](https://tk-pichost-1325224430.cos.ap-chengdu.myqcloud.com/blog/1779416828524.png)

![打包截图2](https://tk-pichost-1325224430.cos.ap-chengdu.myqcloud.com/blog/1779417129655.png)

3. 安装到 VS Code

方式一：命令行

```bash
code --install-extension vscode-acwing-1.0.0.vsix
```

方式二：通过 VS Code 界面

- 打开 VS Code
- 按 `Ctrl+Shift+P`，输入 "Extensions: Install from VSIX..."
- 选择生成的 `.vsix` 文件进行安装

方式三：在文件管理器中右键 `.vsix` 文件直接安装

![安装截图](https://tk-pichost-1325224430.cos.ap-chengdu.myqcloud.com/blog/1779416962556.png)

4. 安装完成后重新加载 VS Code 即可使用
